 # 全タグ付きイメージを一括で削除
gcloud artifacts docker images delete asia-northeast1-docker.pkg.dev/polar-fulcrum-358119/my-docker-repo/my-service --delete-tags --quiet
(Cloud Build 履歴削除: gcloud builds list で一覧確認
 → gcloud builds delete BUILD_ID
 → 全削除するなら gcloud builds list --format="value(ID)" | xargs -n1 gcloud builds delete --quiet)不要
